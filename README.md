# Padland
Padland is a tool to manage, share, remember and read collaborative documents based on the Etherpad technology in Android.

## Current version
0.1

## Changelog
- v0.1 - Added multiple-server support
- v0.1 - Just commit

## Technology
Etherpad depends on:
- jQuery
- etherpad-lite-jquery-plugin
